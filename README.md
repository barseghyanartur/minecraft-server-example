# Minecraft multi-player instructions

## Installation

1. [Grab the MineCraft server](https://github.com/itzg/docker-minecraft-server/)
2. [Grab the TLauncher](https://tlauncher.org/en/install-java.html)

## Wire-up

### Start the server

```shell-script
docker-compose up
```

### Start the client

```shell-script
./TLauncher.sh
```
